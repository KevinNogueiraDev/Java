# Jogo de Adivinhação em Java

## Descrição

Este é um jogo simples desenvolvido em Java no qual o programa gera um número aleatório entre **0 e 100**. O objetivo do jogador é descobrir qual número foi escolhido.

A cada tentativa, o programa informa se o número digitado é maior ou menor que o número secreto, até que o jogador acerte.

## Funcionalidades
- Geração de número aleatório entre 0 e 100.
- Dicas para auxiliar o jogador.
- Contagem do número de tentativas.
- Mensagem de vitória ao acertar.

## Tecnologias utilizadas
- Java
- Maven
- NetBeans

## Como executar
1- Clone este repositório.
2- Abra o projeto no NetBeans.
3- Execute a classe principal.
